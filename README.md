# Student Library Management System

A refactored Object-Oriented PHP application for managing library books and borrow records. This project prepared statements for security.

## Author

* Janelle

## Requirements

* PHP 8.0+
* MySQL
* Git

## Installation

1. Clone repository
2. Import database
3. Configure database in `DatabaseConfig.php`

## Features

* Add books
* View books
* Borrow books
* Return books
* Calculate fines

## Structure

* Config – database and constants
* Entity – data models
* Repository – database queries
* Service – business logic
* Public – entry point
